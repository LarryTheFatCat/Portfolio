# Portfolio

I'm just using this to host my portfolio. It's a simple static site, built with [Jekyll](http://jekyllrb.com/).

## Development

To run the site locally, you'll need to have [Jekyll](http://jekyllrb.com/) installed. Then, run `jekyll serve` from the root directory of the project. The site will be available at [http://localhost:4000](http://localhost:4000).

## Deployment

The site is hosted on [GitHub Pages](https://pages.github.com/). To deploy, simply push to the `master` branch.
